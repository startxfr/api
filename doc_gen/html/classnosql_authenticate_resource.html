Detailed Description

This resource is used to authenticate a user by using a nosql database backend.

Method GET PUT and DELETE are forbidden, consequently the request will be rejected. Use POST. The request should feature a Json object of the following type:

{
"type":"post"
"url":" _BASE_URL_ . 'auth.basic'" < _BASE_URL_ being the root url of the api
"data":
{
"format": "json",
"app": "www",
"login": " _your_login_ ",
"pwd": " _your_pwd_ "
}
}
